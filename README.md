# Housing Price Forecasting

## Overview
Forecasted U.S. housing prices using macroeconomic indicators by applying regression and ARIMA models to analyze trends and evaluate model performance.

## Tools
- STATA
- Excel

## Methods
- Regression
- ARIMA
- Exponential Smoothing

## Key Findings
- ARIMA and regression models outperformed baseline models
- Mortgage rates and unemployment significantly impact housing prices
